Documentación de la API Rest

Bienvenido a la documentación de la API Rest de la aplicación, aquí podrás encontrar las peticiones REST que puedes realizar para obtener los datos en formato JSON sobre la situación del SarsCov2 en Andalucía, estos datos se pueden encontrar en el informe diario del Ministerio de Salud de España o en el portal de IECA.

Para obtener los datos basta con hacer click en alguno de los enlaces, se nos presentarán los datos en un formato como el siguiente: 

HTTP 200 OK
Allow: GET, OPTIONS
Content-Type: application/json
Vary: Accept

[
    {
        "id": 1,
        "name": "Almería",
        "region_name": "Andalucía",
        "poblation": 727945,
        "confirmedPDIA": 54342,
        "totalConfirmed": 54650,
        "tasa14days": 77,
        "tasa7days": 37,
        "deceased": 829,
        "recovered": 52720
    }
]

Donde obtendremos un objeto JSON que podremos mapear en nuestra aplicación o bien obtener con python utilizando scrapping.
Estos datos son de uso público y pertenecen a fuentes oficiales.
